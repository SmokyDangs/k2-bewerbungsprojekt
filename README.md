# YOLO Computer Vision Bewerbungsprojekt

Dieses Repository enthält eine einsatzbereite Web-Applikation zur Objekterkennung basierend auf **YOLOv8/v10**. Das Projekt wurde entwickelt, um die effiziente Integration von Deep-Learning-Modellen in eine benutzerfreundliche Oberfläche zu demonstrieren.

## 🚀 Features
* **Echtzeit-Erkennung:** Schnelle Inferenzzeiten optimiert für CPU/GPU.
* **Custom Model:** Einsatz eines speziell trainierten Modells (`best.pt`) für spezifische Anwendungsfälle.
* **Web-Interface:** Einfacher Upload und Visualisierung der Ergebnisse direkt im Browser.
* **Modularer Code:** Sauber strukturierte `app.py` für leichte Erweiterbarkeit.

## 🛠 Tech Stack
* **Sprache:** Python 3.9+
* **Framework:** Streamlit / Flask (je nach deiner app.py)
* **KI-Modell:** YOLO (Ultralytics)
* **Bildverarbeitung:** OpenCV, Pillow
